Connect and Join The Filipino Minecraft Server

EASY AND PLAY FREE

  1. Open up Minecraft and navigate to the Multiplayer Menu
  2. Click on the "Add Server" button
  3. Enter play.pinoylifecraft.com as the server IP
  4. Make sure to select Server Resource Packs: Enabled
  5. Press Done and Connect to start playing!

You don't need a Minecraft Java account to play Pinoy LifeCraft, we do this so that everyone can play.

Buy Minecraft if you can afford P1364.56, you can purchase one of these from the official website.

Did you encounter problem when joining?

It is possible that you will be unable to connect to the server at any time. There could be a variety of reasons for this. We have a list of problems and a solution you might be having.
