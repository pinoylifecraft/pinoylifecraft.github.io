What is Pinoy LifeCraft?

Pinoy LifeCraft is a massive multiplayer online role playing game enhance Minecraft: Java Edition server. Minecrafters team up with online friends to survive, build, explore, fight the enemies and have fun using their chosen role.

Whether you are building up your base, exploring the server worlds or completing game or daily quests, we have something for everyone here on Pinoy LifeCraft!

SERVER FEATURES

MMORPG

Has 3 roles to choose (Warrior, Archer and Mage). Robust skills with unique skill abilities and follows an XP system of a real RPG/MMO.

New Items

Custom items, weapons and blocks. Furnitures are completely based on a medieval period atmosphere to fit exactly on the server's theme.

Server UI

Customized user interface and menus. No more chest menu thanks to the power of custom resourcepack.

Chat Emoji

Supports sending emoji on game chat. No special code just type :) and it will automatically convert into emoji when message sent.

Your Logo

Show your clan logo to everyone. If your clan is verified you will be able to put your clan logo in your in-game name.

Many Rewards

Get rewards through daily, weekend, quest, crates and killing monsters. Feel free suggest us some events or make your own!

No Resourcepack Conflicts

Resourcepack allow us to customize textures, sounds, interface, font styles and other stuffs. Pinoy LifeCraft is using custom resourcepack to make the server unique, break its limit, add new items and allow us to create truly original server.

Please enable server resourcepack when joining the server.

Unique Filipino Minecraft Server

Thanks to our custom resourcepack we are able to break Minecraft vanilla's limit. By the power of resourcepack your gameplay to the server will be much different compare to classic servers.

We ensure to you that more likely everything is organized!
